API Contact Form Integration

Embed public endpoint scripts in external HTML templates

POST http://127.0.0.1:5000/api/leads

You can capture leads automatically from contact forms on any website. Send form data as a JSON payload to this endpoint. The lead will instantly propagate with status New and source Website.

Example Integration Code (HTML/JS)
<!-- Simple HTML Form -->
<form id="contactForm">
  <input type="text" name="name" placeholder="Full Name" required />
  <input type="email" name="email" placeholder="Email Address" required />
  <input type="text" name="phone" placeholder="Phone Number" />
  <input type="text" name="company" placeholder="Company Name" />
  <textarea name="message" placeholder="Your Message"></textarea>
  <button type="submit">Send Message</button>
</form>

<!-- Javascript Handler -->
<script>
document.getElementById('contactForm').addEventListener('submit', async (e) => {
  e.preventDefault();
  const formData = new FormData(e.target);
  const data = {
    name: formData.get('name'),
    email: formData.get('email'),
    phone: formData.get('phone') || '',
    company: formData.get('company') || '',
    source: 'Website', // Or 'Facebook', 'LinkedIn', etc.
    message: formData.get('message') || ''
  };

  try {
    const res = await fetch('YOUR_BACKEND_URL/api/leads', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data)
    });
    const result = await res.json();
    alert('Lead captured successfully!');
  } catch (err) {
    console.error('Submission failed', err);
  }
});
</script>
